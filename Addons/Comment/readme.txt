评论插件，功能演示版本
使用方法：
1. 将代码解压到Addons
2. 将comment.sql数据表导入数据库(正式版的时候就不用了)。
3. 在后台开启此插件。
4. 下面？下面没了！

功能：
1. 评论
2. 对评论“顶”
3. 用户或匿名均可（匿名必须填写用户名及邮箱）
